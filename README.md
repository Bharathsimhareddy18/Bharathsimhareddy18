## Shipped & Live

**[TradeMinds — Autonomous Paper Trading Agent](https://trademind-agent.vercel.app)**  
3-agent system (Momentum, Scalp, Chat) running live on AWS EC2. Executes NSE trades autonomously, self-determines wake-up intervals based on news sentiment, and answers natural language queries about its own decisions.  
`FastAPI` `APScheduler` `GPT-4o-mini` `Tool Calling` `Supabase` `AWS EC2`

**[Indian Law Assistant — Hybrid RAG + LLMOps](https://bharath-reddy-ai-indian-law-rag-eval.hf.space/docs)**  
Hybrid BM25 + FAISS retrieval with LLM router, HyDE query expansion, and full LLMOps observability — logging faithfulness, correctness, and context relevance per query. UI under development.  
`FastAPI` `BM25` `FAISS` `LLM-as-Judge` `Supabase` `Hugging Face Spaces`

**[Career AI — Resume-JD Intelligence Engine](https://career-ai-flax.vercel.app)**
Parallelized LLM calls via asyncio.gather, LLM-based semantic skill extraction + cosine similarity. Deployed on AWS ECS.  
`FastAPI` `asyncio` `Pydantic` `Docker` `AWS ECS`
